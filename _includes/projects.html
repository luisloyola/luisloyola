Proyectos Universitarios

Detección de comportamiento - Sistema que entrega información cuantitativa y estadística sobre el rendimiento de los conductores de Subus.
Diapositivas interactivas desde XML - Sistema para crear diapositivas interactivas a partir de un archivo XML, facilitando la creación de animaciones, y uso de voz.
File-Splitter - Divisor y recuperador de archivos usando pipes. Basado en RAID 5. Desarrollado para el curso de Sistemas Operativos.
Revisador de pruebas - GoCheck es una aplicación que automatiza y facilita el trabajo de los profesores para la revisión de pruebas.
Hexxagon Clone - Clon del clásico juego de DOS llamado Hexxagon en MIPS ensamblador.
